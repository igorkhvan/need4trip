UX COPY CONTRACT — Club Creation (v1.1)

Status: CANONICAL  
Scope: Paywall + Purchase + Post-payment + Create Club  
Policy: 1 subscription → 1 club  
Validity rule: Subscription is valid for ONE MONTH from the date of purchase  
Paywall reason: CLUB_CREATION_REQUIRES_PLAN  
Tone: Clear, neutral, non-punitive  
Language: EN (RU versions must be semantic equivalents)

────────────────────────────────────────────
1. PAYWALL — BEFORE PURCHASE
Reason: CLUB_CREATION_REQUIRES_PLAN
────────────────────────────────────────────

Title:
Create a club

Primary message:
A club requires an active subscription.

Supporting message:
Each subscription gives you the right to create one club.  
The subscription is valid for one month from the date of purchase.

Primary CTA:
Get subscription

Secondary CTA (optional):
Learn more

Copy rules:
- Do NOT use words like: error, failed, denied
- Do NOT mention one-off credits or upgrades
- Do NOT create urgency or pressure
- Message must be informative, not persuasive

────────────────────────────────────────────
2. PURCHASE CONFIRMATION (POST-PAYMENT)
(NOT club creation)
────────────────────────────────────────────

Title:
Subscription activated

Message:
Your subscription is now active and valid for one month from today.  
You can create your club at any time during this period.

Helper note (optional, smaller text):
The subscription will be linked to the club once it’s created.

Primary CTA:
Create club

Rules:
- Do NOT imply that a club has already been created
- Do NOT auto-create a club
- This screen confirms entitlement only

────────────────────────────────────────────
3. DEFERRED ACTION (USER LEAVES AFTER PURCHASE)
(No dedicated screen — consistency rule)
────────────────────────────────────────────

Implicit copy invariant:
- Subscription validity must always be stated as
  “valid for one month from purchase”
- Never imply immediate action is required
- Never imply that delaying is dangerous or punished

────────────────────────────────────────────
4. CREATE CLUB FORM (S2 — SUBSCRIPTION ACTIVE, CLUB NOT CREATED)
────────────────────────────────────────────

Helper text (optional):
This subscription is valid for one month from purchase  
and will be linked to this club after creation.

Rules:
- No warnings
- No urgency
- No alternative actions

────────────────────────────────────────────
5. ATTEMPT TO CREATE SECOND CLUB (S3)
────────────────────────────────────────────

Paywall title:
Create another club

Message:
Each club requires a separate subscription.  
Subscriptions are valid for one month from purchase.

Primary CTA:
Get another subscription

Rules:
- Same paywall mechanics
- Same reason code
- Different title/message only

────────────────────────────────────────────
6. COPY GUARDRAILS (MANDATORY)
────────────────────────────────────────────

Allowed:
- “Valid for one month from purchase”
- “You can create your club at any time during this period”
- “Each subscription gives the right to create one club”

Forbidden:
- Countdown timers
- “Hurry up”, “Last chance”, “Limited time”
- Fear-based or pressure language
- Legal or contractual wording

────────────────────────────────────────────
7. SINGLE SOURCE FORMULA (MUST NOT DIVERGE)
────────────────────────────────────────────

Subscription gives you the right to create one club  
and is valid for one month from the date of purchase.

This sentence is the semantic anchor.
All variations must preserve this meaning exactly.

────────────────────────────────────────────
END OF UX COPY CONTRACT v1.1
